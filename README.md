# SkillSwap

SkillSwap is a peer-to-peer skill exchange platform where users trade skills instead of money. Users can offer skills they know and learn skills they want by connecting with others through the platform.

---

## Features

* Skill-for-skill exchange (no money involved)
*  User profiles with offered & requested skills
*  Smart skill matching
* Chat / session coordination (planned)
* Modern frontend with scalable backend

---

## Tech Stack
